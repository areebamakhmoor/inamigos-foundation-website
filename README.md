# 🌍 InAmigos Foundation Website

A modern, responsive NGO awareness website developed during my **Web Development Internship at InAmigos Foundation**. The website aims to create awareness about the foundation's mission, showcase its initiatives, and encourage community participation through a clean, user-friendly interface.

## 🔗 Live Demo

👉 https://areebamakhmoor.github.io/inamigos-foundation-website/

---

## 📌 Project Overview

This project was built as part of my internship at **InAmigos Foundation** to design and develop an informative NGO website with a responsive layout and smooth user experience.

The website focuses on presenting the organization's mission, projects, and impact while ensuring accessibility across desktop and mobile devices.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
- ⚡ Smooth Scroll Animations
- 📖 About the Foundation Section
- 💙 NGO Mission & Vision
- 📌 Project Showcase
- 📞 Contact Section
- 🚀 Optimized Layout
- 🌐 Cross-Browser Compatible
